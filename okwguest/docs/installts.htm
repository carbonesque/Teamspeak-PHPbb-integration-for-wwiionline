First, download and run the TeamSpeak setup file. www.goteamspeak.com  Once installed, Run Teamspeak.

You will see a screen like this.

 

 

Although this guide was created for the Windows version of Teamspeak, the Mac OSX Client for TeamSpeak may be downloaded here.

 

  
Click on the Settings menu, then click Options.  
Note: Direct Sound will not work with WWIIOL.

You must use Wave as your sound driver.Be sure you have your specific sound hardware selected instead of primary sound driver.

This is all you need to do in this window. Click OK when you are done.

 

 

Click on the 'Other' tab.  If you notice stutters while using TS in-game, set the Process Priority from Normal to High.

If you have a Microsoft GameVoice external "puck", you may enable TeamSpeak support for it here. The other settings are optional.  After you are finished, click OK.

 
From the Settings menu, select Sound Input/Output Settings.  

Be sure you have Push to talk selected instead of Voice Activation.  Voice Activation tends to pick up stray sounds, even your breathing. This can annoy other players. Pick a key to use for transmitting. ALT, and WIN work well. (Space bar and CAPSLOCK do not) You can even use your mouse buttons to transmit. Click on Set, then hit the key.  If you happen to have the Microsoft GameVoice controller, you can use it for your push-to-talk key instead.

You can test your own transmissions here by clicking Activate local test mode, then pressing your transmit key and speaking.

Click Close when you are done.

  
TO CONNECT:  
From the Settings menu, select Connect.  

You will be presented with the Connect To Server window. 

This feature is a great improvement over the old TeamSpeak.  You can save servers by name as well as all of your account information, passwords and your default channel/password.

  
Right-click on Servers. Then select Add Server.  

Type in any label you like in the Label field.  Type in the server's address in the Server Address field.  The Nickname field is what displays on the server. It is not your login name.

For the OKW server, use this syntax for your name:

Squaddies: "-Squad Tag- Name"
GHC Officers: "Name -GHC rank- "

Make sure the Registered option is selected instead of Anonymous.  In the Login Name field, use the login name you registered with.  Check the Auto-Reconnect box; you'll be glad you did. 

You may enter the channel and subchannel you frequently use in the Default Channel and Default Subchannel fields along with the password if there is one. Otherwise leave those fields blank.

Click Connect.

  

You should now be connected to the server.  You should see a screen similar to this. 

A little tip on the letters:

User Flags:

U = Unregistered
R = Registered
CA = Channel Admin
SA = Server Admin

Channel Flags:

D = Default Channel
R = Registered (permanent)
S = Subchannels Available
P = Password Protected
M = Moderated
