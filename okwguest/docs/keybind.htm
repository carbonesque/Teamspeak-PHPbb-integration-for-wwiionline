The universal keybind will be the greatest benefit of this server design. Downloading and installing this keybind will allow any player to switch to any channel on the server as well as adjust TS volume & mute, whisper bindings, and channel commander whispers (CCWs). Most players will only need to know a few keys to move around their own channels. The keybind is easy to learn however, and most commands can be executed with only one hand.

Included is a link to the keybind system described below. Squads are welcome to modify this file for their needs and distribute it to their players. Automatic channel password entry can be added to the keybinds as well. The “/” and “*” keys on the numpad are to be used as "shift" or "ctrl" keys used in conjunction with a number. “/” + num-key will switch you to a parent channel while “*” + num-key will switch you to the subchannel of that parent channel. This server design has 112 channels (Less than half of the present server) and only 18 keys are used to switch to those channels.

From your Teamspeak program, click on the Settings menu, then select Keysettings.

Now you will see the Keybindings dialog box.

Click the Import button.

Browse to the keybind file you downloaded (filename should be similar to "03.01.04.keybind.kbn")

Once you import your keybind, you will be able to traverse channels, adjust volume, use whispers, etc.

Miscellaneous Commands:

Scroll Lock = Toggle Channel Commander
F10 = Whisper/broadcast to all Channel Commanders on the server. (use sparingly)
F11 = Whisper to all Channel Commanders in your Korps/Luftflotte/Seeflotte
F6 = Toggle Mute Output
F7 = Toggle Away

Lower Volume 10%:

R Shift + “-“
Num 0 + Left Arrow

Raise Volume 10%:

R Shift + “+“
Num 0 + Right Arrow

L CTRL + F1 = Bind last whisperer to preset 1
L CTRL + F2 = Bind last whisperer to preset 2
L CTRL + F3 = Bind last whisperer to preset 3

F1 = Whisper to preset 1
F2 = Whisper to preset 2
F3 = Whisper to preset 3

Switch to Previous Channel

Numpad / + Numpad -
Numpad * + Numpad -
Numpad 0 + Up arrow
R CTRL + “-“

Switch to Next Channel

Numpad / + Numpad +
Numpad * + Numpad +
Numpad 0 + Down arrow
R CTRL + “+“
